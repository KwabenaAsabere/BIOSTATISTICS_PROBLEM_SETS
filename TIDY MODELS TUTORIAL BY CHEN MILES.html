A simple recipe

simple_res <- ames_train %>% 
  recipe(Sale_Price ~ Neighborhood + Gr_Liv_Area + Year_Built + Bldg_Type,data = .) %>% 
  step_log(Gr_Liv_Area,base = 10) %>% 
  step_dummy(all_nominal_predictors())

simple_res

We can add recipe objects to workflow objects. If a formula (which serves as a preprocessor) already exists,you must remove it before starting a new recipe.

lm_workflow <- 
  lm_workflow %>% 
  remove_variables() %>% 
  add_recipe(simple_res)

Interaction Terms

Always make dummy variables first before adding interaction points

 simple_res <- ames_train %>% 
  recipe(Sale_Price ~ Neighborhood + Gr_Liv_Area + Year_Built + Bldg_Type,data = .) %>% 
  step_log(Gr_Liv_Area,base = 10) %>% 
  step_dummy(all_nominal_predictors()) %>% 
  step_interact(~Gr_Liv_Area:starts_with("Bldg_Type_"))

Spline Terms

simple_res <- ames_train %>% 
  recipe(Sale_Price ~ Neighborhood + Gr_Liv_Area + Year_Built + Bldg_Type,data = .) %>%

  step_dummy(all_nominal_predictors()) %>% 
  step_interact(~Gr_Liv_Area:starts_with("Bldg_Type_")) %>%
